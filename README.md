## Git Commands

Commands list 

- git add
- git clone 
- git init 
- git status 
- git commit 
- git log 
- git push 
- git pull
- git ignore 
- git branch 

Sl no | git commands | Description |
----- |-------------- |------------ |
1 | git add |Adds files to the staging area before committing |
2 | git clone | Copies a remote repository to your local system |
3 | git init | Initializes a new Git repository in a folder |
4 | git status | Shows the current state of files |
5 | git commit | Saves staged changes to the repository with a message |
6 | git log | Displays the commit history of the repository |
7 | git push | Uploads local commits to a remote repository |
8 | git pull | Fetches and merges changes from a remote repository |
9 | git ignore | Specifies files and folders that Git should ignore |
10 | git branch | Creates, lists, or deletes branches in a repository |
